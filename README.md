# Literary Nexus Event Landing Page

## Project Overview

Literary Nexus is a responsive event landing page created for a fictional literary event. The website provides information about the event, featured authors, event countdown, and RSVP registration.

The project was developed using HTML, CSS, and JavaScript with a focus on responsive design and user-friendly interaction.

## Features

* Responsive landing page design
* Hero section with event information
* Live countdown timer
* Featured authors section
* RSVP registration form
* Smooth scrolling navigation
* Mobile-friendly layout
* Dynamic data fetching using APIs
* Lazy loading for images

---

## Functionalities Implemented

### Countdown Timer

Displays the remaining time until the Literary Nexus event begins.

### Author Section

Fetches author information dynamically and displays featured speakers.

### RSVP Form

Allows users to submit their registration details for the event.

### Smooth Navigation

Provides seamless scrolling between sections of the webpage.
